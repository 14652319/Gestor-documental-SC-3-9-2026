⚠️ ADVERTENCIA CRÍTICA

Esta funcionalidad elimina PERMANENTEMENTE datos de la base de datos. La acción es IRREVERSIBLE. Se enviará un código de validación por correo electrónico para confirmar la operación.

🔐 Proceso de Eliminación Segura:

  1. Seleccione el tipo de rango y las fechas
  2. Click en "Solicitar Eliminación" - se generará un código de 6 dígitos
  3. Revise su correo electrónico y copie el código recibido
  4. Ingrese el código en el formulario de confirmación
  5. La eliminación se ejecutará automáticamente al validar el código
